grid-chen

Very lightweight and fast editable web grid with strict MS-Excel adherence to user experience. Very modern (web components, es6 modules) with no dependencies. For documentation see grid-chen Repository

Supported Column Data Types

There are five natural types to express a matrix structure. All of them are supported by default.

  1. Array of Row Arrays
  2. Array of Row Objects
  3. Array of Column Arrays
  4. Object of Columns
  5. Single Column Array
