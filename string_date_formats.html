gridchen Index / String Date Formats

Supported date formats are full-date, date-partial-time and date-time. The formats date-partial-time and date-time can have an additional period property, which can have values in ['DAYS', 'HOURS', 'MINUTES', 'SECONDS', 'MILLISECONDS']. Dates are displayed and copied to clipboard in ISO format with local time zone offset. Edit mode allows to input dates in the default locale, for example 27.10.2019 for de-locale.
